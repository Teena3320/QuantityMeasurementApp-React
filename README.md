# Quantity Measurement App

A full‑stack **Quantity Measurement & Conversion Application** built using **Spring Boot (Backend)** and **React (Frontend)**.

The app supports **unit conversion, comparison, arithmetic operations**, and **operation history tracking** across multiple measurement types such as **Length, Volume, Weight, and Temperature**.

---

## 🚀 Features

### ✅ Supported Measurement Types
- Length
- Volume
- Weight
- Temperature

### ✅ Operations
- Convert units
- Compare quantities
- Add quantities
- Subtract quantities
- Divide quantities

### ✅ Key Capabilities
- Instant conversion on unit/value change
- Cross‑unit arithmetic with automatic base‑unit conversion
- Strong backend validation (measurement type ↔ unit consistency)
- Persistent operation history stored in database
- Error tracking for invalid operations
- Clean, responsive UI

---

## 🏗️ Tech Stack

### Backend
- Java 17+
- Spring Boot
- Spring Web
- Spring Security (configured for REST APIs)
- Bean Validation (Jakarta Validation)
- JPA / Hibernate
- H2 / PostgreSQL (configurable)

### Frontend
- React
- JavaScript (ES6+)
- Fetch API
